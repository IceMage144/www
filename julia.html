How to use

  • Enter an imaginary number at the fields to see the corresponding Julia set on the left canvas
  • You can hold the Z key and point with the mouse to the Mandelbrot set (on the right canvas), so you can see the Julia set of that point on the left canvas
  • You can increase or decrease the resolution using the "+" and "-" buttons
