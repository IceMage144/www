How to use

 
  • Put a formula in the first line
  • Put the variable you want to derivate in the second line
  • Click on the "Calculate" button and see the derivative!

-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-

Valid functions and variables

 
  • Variables can be an upper or lower case letter (excluding "e"), and can have subscript using "_", ex.: x_i
  • To use unary "-", use "~" or it will show an error message
  • You can use the constants e, pi and gr (golden ratio)
  • pow(a,b) = a^b (Obs.: sqrt not implemented yet! Use pow(a,1/2) to get sqrt(a))
  • log and ln were implemented too
  • Every trigonometric function were implemented (inverses, inverse functions and hiperbolic)
  • sum(i, n, a_j) = summation of a_i's, with j going from i to n (Obs.: don't try to derivate it in a_j!)
  • Maybe in the future, factorial can be implemented
  • Maybe in the future I'll try to take off some zeroes and ones from the result too :)
